# Linker Framework

PHP Framework for Building REST API

### Documentation

See [Documentation](https://eru123.github.io/linker)

# Installation
### Via Composer

```bash
composer require eru123/linker
```

## For Contribution

```bash
git clone https://github.com/eru123/linker.git
cd linker
composer install
```
